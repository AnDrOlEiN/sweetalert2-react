> sweetalert in react


## Install

```
$ npm install sweetalert-react
```


## Usage

```js
var SweetAlert = require('sweetalert-react');

render() {
  return (
	<div>
	  <SweetAlert
	    title="ERROR!!!!"
	    text="some bad thing happened"
	  />
	</div>
  );
}
//=>
```
